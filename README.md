# ALU UVM Testbench

A UVM 1.2 verification environment for an 8-bit ALU built in SystemVerilog.

## What it verifies
- Addition, Subtraction, Bitwise AND, Bitwise OR

## UVM Components
- Transaction, Sequence, Driver, Monitor, Scoreboard

## Result
200/200 checks passing — 0 errors, 0 warnings

## Run it
https://edaplayground.com  — paste files, select Aldec Riviera-PRO, UVM 1.2
