# ALU UVM Testbench

A UVM 1.2 verification environment for an 8-bit ALU built in SystemVerilog.

## What it verifies
- Addition, Subtraction, Bitwise AND, Bitwise OR

## UVM Components
- Transaction, Sequence, Driver, Monitor, Scoreboard with functional coverage
